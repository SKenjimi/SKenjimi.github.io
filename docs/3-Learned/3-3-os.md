# 3-3. Operating Systen --Learned
## Ubuntu
### System locate
1. ロケールパッケージのインストール  
`$ apt -y install language-pack-ja-base language-pack-ja`
1. ロケール設定の一覧表示  
`$ localectl list-locales`
1. ロケールの設定確認  
`$ localectl`
1. ロケールの設定変更  
`$ localectl set-locale LANG=ja_JP.UTF-8 LANGUAGE="ja_JP:ja"`
1. ロケールの設定確認  
`$ localectl`
