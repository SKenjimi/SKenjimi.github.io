# 3-6-1. Azure create VM
## セットアップ
### Azure CLIで作成する
1. Cloud Shell から次の az vm create コマンドを実行して、Linux VM を作成
<pre>
az vm create \
  --resource-group learn-bffa8940-a7ec-4334-83d7-29fd23dc0794 \
  --name my-vm \
  --image UbuntuLTS \
  --admin-username azureuser \
  --generate-ssh-keys
</pre>
2. test
